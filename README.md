# Итоговый проект первого года обучения  
## Бриф 1. Агентство недвижимости

В данном проекте реализован сервис прогнозирования стоимости недвижимости на основе машинного обучения.

Модель обучена на датасете, содержащем сведения о ~377 000 объектах недвижимости, выставленных на продажу в США.

Проект включает:
- анализ и подготовку данных,
- обучение модели,
- развёртывание модели в виде API-сервиса с использованием FastAPI и Docker.

---

## Данные

Исходный датасет доступен по ссылке:  
https://drive.google.com/file/d/11-ZNNIdcQ7TbT8Y0nsQ3Q0eiYQP__NIW/view

Данные содержат информацию об объектах недвижимости и их ценах.

---

## Этапы работы

1. Первичный анализ данных, очистка и отбор признаков  
2. Подготовка признаков и обучение моделей  
3. Выбор финальной модели и оценка качества  
4. Подготовка сервиса для использования в продакшене  

---

## Модель и обработка данных

- Для предварительной обработки данных реализована функция `_do_preprocessing()`
- Категориальные признаки кодируются с помощью `OneHotEncoder`
- В качестве финальной модели использован алгоритм **CatBoostRegressor**

## Реализация сервиса

Сервис реализован в виде REST API с использованием:
- **FastAPI**
- **Docker**

После запуска сервис предоставляет интерфейс для ввода параметров объекта недвижимости и получения прогнозной цены.

Swagger-документация доступна по адресу:
http://localhost:8000/docs
